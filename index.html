Analysis pipeline

From URL to insight

Eight steps run across three parallel Celery workers, each step streaming live progress to your browser.

01
Ingest
Clone via gitpython. Filter by extension. Cap at 300 files.
›
02
Parse
tree-sitter extracts imports, exports, functions per file.
›
03
Graph
Resolve imports to absolute paths. Write edges to Neo4j.
›
04
Classify
Ollama assigns each file an architecture tier: entry · logic · util · config.
›
05
Score
NetworkX PageRank computes impact score. Positions nodes in 3D.
›
06
Summarise
Ollama round-robins across 3 GPUs. 1-2 sentence summary per file.
›
07
Embed
nomic-embed-text creates vectors for all summaries. RAG index built.
›
08
Onboard
Claude Haiku generates the prioritised reading guide. Done.
